## Go data-types summary

* Statically Typed: Once a variable has a type (like int), it can’t change to another (like string).

* Booleans (bool): Stores true or false. If left empty, the default is always false.

* Integers (int): Use int for whole numbers. Signed allows negatives; Unsigned is for positive numbers only.

* Int Sizes: You can specify size (e.g., int8 to int64) to save memory, but int is the standard default.

* Floats: Use float32 or float64 for decimals. float64 is the default and holds much larger/more precise numbers.

* Strings: Wrapped in double quotes. An unassigned string defaults to an empty value ("").

* Memory Matters: Choosing the right numeric type (like int8 vs int64) helps manage how much space my data takes up.
