## Go Operators
I see operators as the tools I use to manipulate my data, whether I'm doing simple math or complex logic. I can use them to combine raw values, mix variables with values, or even stack variables on top of each other to get a final result.

I generally group them into five main categories based on what they can be used to achieve:

* Arithmetic Operators: These are my go-to for math. I use +, -, *, and / for basic calculations, and % (modulus) when I need to find the remainder of a division.

* Assignment Operators: I use these to put values into variables. Beyond the standard =, I often use compound ones like += or -= to update a variable’s value based on its current state without retyping its name.

* Comparison Operators: When I need to make decisions, I use these to see how values stack up. == checks if they're equal, while > or < help me determine which is larger. These always hand me back a boolean (true or false).

* Logical Operators: I use these to combine multiple conditions. && (AND) requires everything to be true, || (OR) needs just one part to be true, and ! (NOT) lets me flip a true to a false.

* Bitwise Operators: I rarely use these unless I'm doing low-level programming. They let me manipulate the individual bits (1s and 0s) of a number directly using operations like & (AND) or | (OR).
